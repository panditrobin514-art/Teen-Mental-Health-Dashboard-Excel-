# 📊 Teen Social Media & Mental Health — Excel Dashboard

## 📌 Overview
An end-to-end Excel dashboard built to analyze how social media usage 
impacts the mental health of 1,200 teenagers (ages 13–19).  
Covers sleep patterns, anxiety, stress, addiction levels, platform behavior, 
and risk segmentation — all visualized in a single interactive dashboard.

---

## 🗂️ Sheets Included

| Sheet | What's Inside |
|-------|--------------|
| **Dashboard** | 6 charts + 8 KPI cards — full visual summary |
| **Usage & Wellbeing** | Social media hours vs sleep, stress, anxiety, addiction |
| **Demographics** | Age group, gender, platform breakdown |
| **Risk Segments** | At-risk population sizing and profiling |
| **Business Insights** | Strategic recommendations |
| **Raw Data** | Complete 1,200 record dataset |

---

## 🔍 Key Findings

- 📌 **64.8%** of teens show at least one elevated risk indicator
- 📌 **TikTok** users report highest anxiety (5.75) and addiction (5.62) scores
- 📌 **Ages 15–16** peak on anxiety, stress AND addiction — most critical age window
- 📌 Heavy users **(6h+/day)** show **67.3% at-risk rate** vs 62.2% for light users
- 📌 Sleep stays flat at ~6.45h regardless of usage — screen time alone doesn't explain sleep loss
